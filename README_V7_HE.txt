Market Open Analyzer V7 – Engineering Patterns

הפעלה מקומית:
cd ~/Desktop/market_open_analyzer
source venv/bin/activate
streamlit run market_open_analyzer_v7_engineering_patterns.py

העלאה ל-Streamlit Cloud:
1. העלה את market_open_analyzer_v7_engineering_patterns.py ואת requirements.txt לאותה תיקייה ב-GitHub.
2. הגדר Main file path ל-market_open_analyzer_v7_engineering_patterns.py.
3. בצע Reboot app.

הגדרות התחלה מומלצות:
- סוג השקעה: חצי שעה
- מינימום ניקוד: 8
- מקסימום עסקאות חדשות בסריקה: 2
- מקסימום פתוחות: 4
- מנוע תבניות: פעיל
- מינימום תבניות: 10
- ביטחון מינימלי: 0.58
- ביטחון חזק: 0.70

בדמו בלבד. יש לאסוף לפחות 50-100 עסקאות לפני שינוי ספים משמעותי.
